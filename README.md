# Smart Food Rescue System

A full-stack MERN application that connects food donors with NGOs or receivers in real-time.

## Features
- Real-time location-based matching and notifications
- Secure JWT Authentication
- Image uploads via Cloudinary
- Socket.io for live updates
- Interactive Maps using Leaflet

## Quick Start

### Prerequisites
- Node.js
- MongoDB

### Setup

1. **Clone the repository**

2. **Backend Setup**
```bash
cd backend
npm install
npm run dev
```
Make sure to copy `.env.example` to `.env` and fill in your details.

3. **Frontend Setup**
```bash
cd frontend
npm install
npm run dev
```

## Tech Stack
- MongoDB, Express, React, Node.js (MERN)
- Tailwind CSS
- Socket.io
- Vite
